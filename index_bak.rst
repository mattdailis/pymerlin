pymerlin |release|
=================

pymerlin is a discrete event simulation framework, built for use in the `Aerie <https://github.com/NASA-AMMOS/aerie>`_ ecosystem.

To learn more about Aerie, read the `Aerie Docs <https://nasa-ammos.github.io/aerie-docs>`_

Ready to get started? Check out the :doc:`Quickstart<quickstart>` guide.

Source code
-----------

You can access the source code at: https://github.com/mattdailis/pymerlin
